# X AI Reply - X 智能回复助手

一个 Tampermonkey 用户脚本，使用 AI 为 X (Twitter) 推文生成智能回复。

## ✨ 功能特性

### 🤖 AI 回复生成
- 支持多种 AI 服务商：OpenAI、Anthropic Claude、Google Gemini
- 自定义 API 地址支持（兼容第三方代理）
- 一键生成多条候选回复

### ⚙️ 灵活配置
- **回复数量**：1/2/3/5 条
- **回复长度**：短/中/长
- **回复风格**：吸引关注、幽默搞笑、专业严谨、犀利毒舌、暖心治愈
- **回复策略**：默认、同意热评、提出新观点、平衡讨论、反驳观点
- **语言选择**：自动检测、中文、英文、日文、韩文
- 支持自定义添加/删除风格和策略

### 📊 评论区分析
- 自动抓取推文评论区热门回复
- AI 分析评论区主要观点和情绪
- 根据分析结果生成更符合讨论氛围的回复

### 🌐 自动翻译
- 检测非中文回复自动翻译成中文
- 并行翻译，快速显示结果

### 📜 历史记录
- 自动保存每条推文的生成记录
- 重新打开面板可查看历史回复
- 最多保留 50 条推文记录

### 💾 设置持久化
- 自动记住上次选择的设置
- 学习用户编辑模式，优化生成风格

## 📦 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击安装脚本或手动添加 `x_ai_reply.user.js`
3. 在 X.com 刷新页面

## ⚙️ 配置

1. 点击 Tampermonkey 菜单中的「⚙️ AI Reply Settings」
2. 选择 AI 服务商（OpenAI/Anthropic/Gemini）
3. 填入 API 密钥
4. 选择模型
5. 保存设置

## 🎯 使用方法

1. 浏览 X.com 推文
2. 点击推文下方的「🤖 AI」按钮
3. 在设置面板中：
   - 选择回复数量、长度、风格
   - （可选）点击「📊 分析评论区热评」
   - 选择回复策略
4. 点击「✨ 生成回复」
5. 在结果中选择满意的回复，自动填入回复框
